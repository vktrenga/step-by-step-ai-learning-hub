# 🤖 Step-by-Step AI Learning Hub

A comprehensive guide to Artificial Intelligence, Natural Language Processing, and Large Language Model frameworks.

---

## 📚 Learning Modules

### 1. 🔤 [Natural Language Processing (NLP)](nlp/readme.md)
Master the fundamentals of NLP including:
- **Text Processing**: Tokenization, stemming, lemmatization, n-grams
- **Core NLP Concepts**: Understanding language, feature extraction
- **Text Analysis**: Sentiment analysis, classification, entity recognition
- **Practical Applications**: Building NLP pipelines and models

**Start here to understand how machines process and analyze human language.**

---

### 2. 🧠 [LLM & Hugging Face](llm_and_huggingface/readme.md)
Explore advanced AI models from Hugging Face:
- **LLM Fundamentals**: RNNs, Attention Mechanisms, Transformers
- **Hugging Face Ecosystem**: Pre-trained models, transformers library
- **Model Selection**: Finding and using the right model for your task
- **Practical Integration**: Loading and deploying models

**Dive into cutting-edge language models and learn how to leverage them.**

---

### 3. ⛓️ [LangChain Framework](langchain/readme.md)
Build production-ready AI applications using LangChain:
- **Framework Basics**: What LangChain is and why it matters
- **OpenAI Integration**: Connecting to powerful language models
- **Prompt Engineering**: Designing effective prompts
- **Runnable Abstractions**: Piping, sequencing, and parallel execution
- **Real-World Use Cases**: Building travel assistants, chatbots, and more

**Learn how to compose AI components into sophisticated applications.**

---

### 4. 🧬 [LangGraph](longgraph/readme.md)
Advanced agent and workflow orchestration:
- **Graph-based Workflows**: Building complex execution flows
- **Agent Architecture**: Multi-agent systems and coordination
- **State Management**: Managing application state
- **Advanced Patterns**: Loop handling, error recovery

**Master sophisticated multi-agent architectures and graph-based workflows.**

---

### 5. 🗄️ [Vector Databases](vectordb/readme.md)
Learn vector storage and semantic search:
- **Vector Embeddings**: Understanding embeddings and similarity
- **Database Fundamentals**: Storing and querying vectors
- **Semantic Search**: Building search systems
- **RAG Applications**: Retrieval-augmented generation

**Implement scalable semantic search and RAG systems.**

---

### 6. 🎤 [Speech Recognition](speech_recognition/readme.md)
Convert speech to text and build voice applications:
- **Audio Processing**: Working with audio files and streams
- **Speech-to-Text**: Transcription models and pipelines
- **Voice Applications**: Building voice-enabled systems
- **Integration**: Combining speech with NLP and LLMs

**Create voice-enabled AI applications.**

---

### 7. 🎨 [Streamlit](streamlit/readme.md)
Build interactive web applications with Python:
- **UI Components**: Buttons, inputs, displays
- **Data Visualization**: Charts and interactive plots
- **Application Deployment**: Sharing and hosting apps
- **Integration**: Combining with AI models and backends

**Deploy AI applications as interactive web interfaces.**

---

### 8. 📋 [Use Cases & Projects](usecases/readme.md)
Real-world applications and end-to-end projects:
- **Transport & Booking**: Multi-agent travel planning systems
- **Chatbots**: Conversational AI systems
- **Data Processing**: Workflow automation
- **Integration**: Combining all technologies

**See practical implementations of all concepts.**

---

## 🚀 Getting Started

1. **Clone or download** this repository
2. **Install dependencies**: `pip install -r requirements.txt`
3. **Choose your learning path** based on your experience level
4. **Explore the notebooks** in the `learning/` directory
5. **Follow the guides** in each module's readme
